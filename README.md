# walpaperr



## Getting Started

Download, install requirements (via `pip` for instance) and run `script.py`. 

### Prerequisites

Basically
* scikit-image
* opencv
* requests

See [requirements.txt](requirements.txt) for full `pip freeze > requirements.txt` output.
